## 项目介绍

### 应用简介

一款可以在安卓智能手表和wearOS手表上面运行的化学学习类软件，除了最基本的元素信息查询，还支持离子，有机基团等多种数据查询

### 立项的原因？

这个项目是去年(2022年)十月底立项的，最开始的时候其实是想方便自己的学习罢。我在去年的8月份购入了人生中的第一块智能手表：OPPO Watch 2，在11月购入了第二块智能手表：OPPO Watch 3Pro，这才有了后面的一切。

最开始我也就是一名普通的智能手表用户，~~毕竟是学生，所以看看视频刷刷酷安很正常好吧~~，但是随着对智能手表的深入了解，我逐渐意识到，安卓智能手表的生态真的是非常贫瘠（和隔壁AppleWatch相比）；

这也是为什么，穿戴圈的个人开发者如此之多的原因之一，~~要是安卓智能手表生态好软件足够好用谁他妈自己做软件~~，所以说在受到一天晚自习的启示之后，我在那周周末回家之后就立下了这个项目


### 为什么而做？

很简单，算是自己的热爱吧，~~虽然我真的很菜~~

这个项目是使用iApp编写的，在这里要感谢冬日，是他带我入坑了这门语言，也是他给了我很多帮助，真的很感谢感谢冬日喵~

这个项目刚立项的时候就确定了`全免费`，算是为爱发电吧，因为我那个时候还有大把资本可以拿来为爱发电
